# CS340-Client-Server-Development
For this course, I focused on keeping my code simple and organized so I could reuse it later. In Project One, I created a CRUD Python module, and in Project Two I used that same module to connect my dashboard to the database. This made my code easier to manage and saved time since I didn’t have to rewrite everything. It also helped keep my program more readable and flexible.

When working through the project, I broke the problem down into steps. I first made sure the database connection worked, then focused on getting the dashboard to display the correct data. This project felt more realistic than previous assignments because I had to think about how everything worked together, not just getting code to run.

Computer scientists build systems that help people work more efficiently. In this case, the dashboard I created would help a company like Grazioso Salvare quickly find and use their data instead of searching manually. Overall, this project showed me how combining a database with a user interface can make information much easier to use.
